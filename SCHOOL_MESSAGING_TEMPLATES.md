### Handling Questions (Common Q&A)

**Q: "How much does it cost?"**
```
Right now, it's completely free. We're in our growth phase and focused on getting RETEC into as many schools as possible to help Ghana's students. Our goal is impact first, monetization later - if ever for schools.
```

**Q: "Who built this?"**
```
We're a Ghana-based team that saw the gap between what students learn and why they learn it. We combined GES curriculum mapping with real career requirements, using Ghana-specific examples throughout. It's built BY Ghanaians FOR Ghanaians.
```

**Q: "What if students choose the wrong career?"**
```
Great question! The beauty is they can search ANY career, anytime. If they change their mind from Teacher to Engineer, they just search again and adjust their focus. It's flexible. Plus, we show related careers so students discover options they hadn't considered.
```

**Q: "How is this different from regular career talks?"**
```
Career talks are great for inspiration, but they usually end with "Study hard in school." RETEC goes deeper: "If you want THIS career, prioritize THESE specific topics for THESE specific reasons." It connects the talk to the classroom.
```

**Q: "Will teachers need training?"**
```
No formal training needed. It's just a website - if you can use Google, you can use RETEC. We provide a simple one-page guide, but honestly, students figure it out in 2 minutes. Teachers can use it to pull examples for lessons or just share the link with students.
```
